# Shell Redirections

This directory contains shell scripting exercises that demonstrate input/output redirection, file manipulation, and basic stream handling in the shell.

How to run

- Make a script executable: `chmod +x <script>`
- Run with: `./<script>` or `bash <script>`

Exercises

- `0-hello_world`: Print "Hello World" to stdout.
- `1-confused_smiley`: Print a confused smiley or similar output.
- `2-hellofile`: Demonstrate redirecting output to a file.
- `3-twofiles`: Work with two files (create/concatenate/compare).
- `4-lastlines`: Print the last lines of a file (using `tail`).
- `5-firstlines`: Print the first lines of a file (using `head`).
- `6-third_line`: Output the third line of a file.
- `7-file`: Inspect or output file contents/metadata.
- `8-cwd_state`: Show current working directory state and contents.
- `9-duplicate_last_line`: Duplicate or repeat the last line of a file.
- `10-no_more_js`: Remove or filter lines containing "js"/"javascript".

Notes

- These exercises focus on redirection operators (`>`, `>>`, `<`, `2>`) and common text utilities (`cat`, `head`, `tail`, `sed`, `awk`).
